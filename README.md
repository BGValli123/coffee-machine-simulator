# ☕ Coffee Machine Simulator

A Python-based simulation of a coffee machine that processes user orders, checks resources, and handles payments.

## 🚀 Features

* Supports espresso, latte, cappuccino
* Resource management (water, milk, coffee)
* Coin-based payment system
* Calculates and returns change
* Provides system report

## 🛠️ Technologies

* Python

## ▶️ How to Run

```bash
python coffee.py
```
